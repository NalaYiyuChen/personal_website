    Here are some projects I've created:

      Gitlet

      Gitlet is a version control system based off Git. My version implemented the following commands: init, add, commit, remove (rm), log, global-log, find, status, checkout, branch, rm-branch, reset, and merge.

      I would love to share my implementation of this project. However, because Gitlet was a class project, and because course policy prohibits making our code public during and after the class, please email me if you would like to see my implementation.

      Gitlit

      Github is a valuable resource for programmers. However, my Violet Hacks team noted that, when we started using Github, we found its interface somewhat intimidating. As such, we created Gitlit to more gently introduce new users to Github through Achievements. Take a look at our project!

      Virtual Guzheng

      The guzheng, or Chinese table harp, is a traditional Chinese instrument which I have been playing for ten years. I have always wanted to share this beautiful instrument with the world; however, a physical instrument is a bit too expensive to purchase and send to everyone I meet. Fortuntely, creating a virtual version of this amazing harp is free, and you can run your mouse over its strings here.

      Sift

      Sift was created for Cal Hacks 6.0, and the goal my team set for the weekend was simply to each learn a new technology. Sift, or "Styles In Finding Things," is a search engine designed to present information to users based upon their style of learning. Check out our project here.
